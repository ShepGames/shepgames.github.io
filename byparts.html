Intergration by parts

Sometimes intergration cannot be done using conventional methods.

There will be two values multiplied together which causes issue.

∫2xsinx dx is an example of this. We will use integration by parts to find the answer.

We pick a u, and dv, we want u to be the easiest to cancle out. 2x differentiates to 2 so this is better for u.

u = 2x and dv=sinxdx. we integrate dv to get v=-cosx+c (but ignore the c)

we also find du/dx which is 2.

All these values are needed for the next stage which is the formula:

(u)(v)-∫(v)du = (2x)(-cosx)-∫(-cosx)(2)dx

Sometimes you will get an intergral which needs inergrating by parts a second time.

Example:

1)Find ∫2x^2 *cosx dx

u=2x^2, du=4xdx

dv=cosxdx

v=sinx+c

So: (2x^2)(sinx)-∫sinx4xdx

We cannot do the second part so intergrate again:

u=4x, du=4dx

dv=sinxdx

v=-cosx+c

(2x^2)(sinx)-((4x)(-cosx)-∫-cosx4dx)

=2(x^2)sinx+4xcosx-4sinx+c
