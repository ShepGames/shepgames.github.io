Double angle formula

Double angle formulue is to split up trigonometric functions and make them easier to work with

Rules:

sin(A±B) = sinAcosB±sinBcosA

cos(A±B) = cosAcosB ∓ sinAsinB

tan(A±B) = (tanA ± tanB)/(1 ∓ AtanB)

sin2A = 2sinAcosA

cos2A = cos^2(A) - sin^2(A)

tan2A = (2tanA)/(1 - tan^2*(A))

And for intergration:

sin^2(A) = 0.5-0.5cos2A : cos2A = 1-2sin^2(A)

cos^2(A) = 0.5+0.5cos2A : cos2A = 2sin^2(A)-1

These can be really helpful in trig questions, to work out and simplify things.

Example:

1)Show that tan(3β-45)≡(tan3β-1)/(1+tan3β)

The left hand side looks more expandable: LHS

≡(tan3β-tan45)/(1+tan3βtan45)

We know that tan45 is 1

So: (tan3β-1)/(1+tan3β(1))

LHS ≡ RHS Therefore QED
