Object oriented

OOP is a computer programming paradigm which creates objects as part of the code.

Objects are written in classes, and manage all their own variables and functions.

Procedural cannot be used like this as it has a set amount of variables, and would struggle to keep them all managed

Objects can be called into existance, and as many as the user wants (can store)
