Computer Science

Operating systems

There are many types of operating systems. An OS is a low-level software which supports the computer's basic functions like

hardware and peripheral management.

Open source

Open source operating systems, like linux, are operating systems which allow modification of the source code.

Advantages

Dis-advantages

Proprietory

Proprietory is software which you typically buy and cannot modify. Windows is an example of this.

Real time

Operating systems which act in real time. This means there is a time share between applications, which gives

the feeling of multiprocess.

Multi user

Simply an oeprating system which allows lots of users to access it.

Single user

This method allows a super user to control everything in the computer system. This is typically found in networks.

Embedded

An embedded operating system is a small functional operating system built specifically for a certain task. Dishwashers

will have an embedded operating system to control the dishwasher's tasks.
