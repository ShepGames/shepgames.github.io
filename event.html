Computer Science

Event driven

Event driven is where the flow of the proggram is determined by events, such as a button click.

Main loop

Each event is checked in the mainloop and when the event is called it will enter it's event handler.

Call back function

When a function may take a while, it uses a callback function which makes a note to come back when it has finished doing what it is doing.

Event handler

Simply processes inputs such as text boxes, text being present, keystrokes and so on.

Event

Another term for user action.

Suitibility

Event driven code is typically used in graphical user interfaces, and websites. Any system where the user has multiple

choices of buttons to press which have no particular order to press them. On a button click the system must do something.
