Computer Science

Programming paradigms

Data is represented in a computer system as an identifier

Constants

A constant is variable is set at the beginning and does not change throughout the program. This could be values like Pi.

Variables

A variable is a value stored to an identifier, which can be set and changed throughout the program. There are local variables

and global variables. Global variables can be used anywhere in the program, local variables are unique to their subroutine.

Variables

local variables

these are limited to being used in the block od code in which they are declared for example within a particular function

Global variables

these can be used anywhere in your program code.

We need to also use appropriate name identifiers for each variable so we are able to know what each one is for.

We normally classify them in a standard format Snake_case, using the underscore or capitalising the second word, CamelCase.
