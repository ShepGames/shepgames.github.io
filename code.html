Code optimization

Improve code quality and efficiency. A program may be optimized so that it becomes smaller in file size.

This is done by removing unused variables, calculations made more efficient and removing other parts of the code which is unused.

Intermediate

Decomposition

Breaking complex problems of a system into parts that are easier to understand, program and maintain.

Object oriented

Code which creates objects to manage it's own variables and data.

Procedural

Code which reads one line at a time of uor program, it uses the variables in program and is alot easier to understand.

Normal programming languages like C and most of the time Python are procederal.

Event driven

Code which does not run unless it is called upon. Javascript is gets called by HTML to run a function, this is event driven.
