Validating data

validation is a process that check to see if an input value makes scene and is reasonable before it is processed.

validation doesn't check to see if the data has been keyed in accurately this is called verification

Run Time Error

a problem that course while an application is being used. these errors result in the application locking or crashing

Validation Check Techniques

constraints can also be used to create validation

Range Check assesses whether data entered is within a valid minimum-to-maximum range

Length Check assesses how many characters have been entered. some very well-known inputs have limited length for example sms

Presence Check assess if data is present

type check assesses whether the data entered is of the correct data type

Format check assesses whether the data entered is of the correct format
