Computer Science A-level

Databases

A database is a persistant tabular form to store data. Data is split into tables, fields (columns) and records (rows).

Databases are typically maintained using SQL. Databases use primary keys to find infomation, which are unique

identifiers in the code like "keycode".

Keys

Secondary - A key which can represent a record, but is not the primary. It may not be unique so will group multiple records.

Foreign - secondary keys which are used in other databases to link them

.

SQL

SQL is the language you are expected to use for databases. An '*' represents a whole table

SELECT ... - means to select all the fiels listed. You can do all (*) or list with commas

FROM ... - means selecting the ... table

.

SELECT * FROM Employees

The above will select all fields from the table Employee

WHERE ... - is the search criteria. Set it out like an IF statement

.

SELECT EmployeeID FROM Employees

WHERE age = 54

The above will output all the IDs of 54 year olds.

ORDER BY ... - orders the output

.

SELECT EmployeeID FROM Employees

ORDER BY wage

WHERE age = 54

The above will order the output of ID's of 54 year olds by their wage.

UPDATE ... - update the tablename

.

UPDATE animals

SET fleas = FALSE

WHERE treatment = TRUE

The above finds all the animals who have had treatment, and update their flea status to false in the animal table.

DELETE ... - delete from the tablename

.

DELETE animals

WHERE died = TRUE

The above deletes all the records where the animal died.
