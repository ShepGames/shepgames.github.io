Data types

Integer - any whole number (1)

String - a series of alphanumeric characters ("hello")

Char - a single ascii character. It uses less storage than a string ('e')

Float or Real - a decimal number (1.23)

List or array - a series of datatypes all assigned

2D array - like a table, you have rows and columns.

Linked list - items in a list which relates to one another item in the list
