Differentiation is used to find the gradient of a line equation.

To differeniate an equation we must follow rules. The indicies will multiply with the value,

and drop by 1: nx^i --> nix^(i-1). With real values 2x^2 will be 4x

Values without indicies drop the variable, so nx --> n. 5x becomes 5

Values without a variable become 0. so n --> 0. 8 becomes 0

We show differentiation of f(x) as f'(x). with y = , we use dy/dx

Example:

1)differentiate the equation y=2x^3 + 3x^2 + 9x + 1

Using the rules we get

dy/dx = (2*3)x^(3-1) + (3*2)x^(2-1) + 9

= 6x^2 + 6x + 9

When differentated, we can find the gradient of any x co-ordinate position by subbing it in and seeing what it equals.

Chain rule:

y = (u)^n --> dy/dx = u'(n)(u)^(n-1)

y=(2x+1)^4

dy/dx = (2)(4)(2x+1)^3

Product rule:

y=uv

dy/dx = u'v + v'u

y=2xcosx --> dy/dx = 2cosx - 2xsinx

Quotent rule:

y = u/v

dy/dx =(u'v-v'u)/(v)^2

y = 2x/cosx --> dy/dx = ((2)(cosx)-(-sinx)(2x))/(cosx)^2
