The basics

Every program you write has an input, process and output

INPUT-returns the user's input

OUTPUT() or Print()-outputs the string in the brackets

str()-converts the data inside the brackets to a string

int()-converts the data inside the brackets to an integer

int()-converts the data inside the brackets to an integer

STRING valueName = ""-declares a string called valueName

INT valueName = 0-declares a int called valueName

CHAR valueName = 'a'-declares a char called valueName

FLOAT valueName = 1.23-declares a string called valueName

len() or .length()-gets the legnth of a string

Using files will use very simple commands

open-opens a file

read-reads data from a file

write-writes data to the file

close-closes the file

Conditions

IF ... THEN-checks the condition

ELIF ... THEN or ELSE IF ... THEN-checks the condition after the condition above

ELSE-if none of the conditions are met

ENDIF-Shows the end of the if statements

FOR i=0,i<10 THEN-starts at 0 and loops till i=10

NEXT i-Next i value meaning it increases by 1

END FOR-Shows where the for loops

WHILE ...-loops while the condition ... is true

END WHILE-Ends the while loop

DO-do the following process after and read the condition at the bottom

WHILE ...-At the end of the do while to decide whether or not to loop
