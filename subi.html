Intergration by substitution

Another way to intergrate difficult equations is to substitute a different value in.

Sometimes you are given a value, other times you are not. It is difficult to pick a value but with practice it is easier.

Example:

1)Find ∫√(1-x^2)dx

∫√(1-x^2)dx has a 1-somthing squared. This reminds us of the 1-cos^2(x) and 1-sin^2(x). Lets make x = cosu

With this we can make u = arccosx

∫√(1-cos^2(u))dx

∫√(sin^2(u))dx

∫sinudx we want it in terms of du rather than dx, so we can intergrate it.

x = cosu so dx/du = -sinu

∫sinu(-sinu)du = ∫-sin^2(u) du

=(-1/3)(sin^3(u))+c which will be horrible to sub x in. Instead use a trig identity:

∫-(0.5-0.5cos2u) du = ∫0.5cos2u-0.5 du

(0.5/2)sin2u-0.5u+c which is c-0.5u+0.25sin2u

if x^2 = cos^2(u) then 1-x^2 = sin^2(u) thus √(1-x^2) = sinu

c-0.5u-0.25sin2u = c-0.5(arccosx)+√(1-x^2)
