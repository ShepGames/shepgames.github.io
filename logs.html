Logs

log means power. when you have 2^x = 5, it is difficult to find this without a log. Log2(5)=x is how you would work it out

The 2 is the base number. If you get a log question with no base, you can assume it is 10.

Laws of Logs.

2^x * 2^y = 2^(x+y)

2^x / 2^y = 2^(x-y)

(2^x)^n = 2^(xn)

loga(x) + loga(y) = loga(xy)

loga(x) - loga(y) = loga(x/y)

nloga(x) = loga(x^n)

Example:

1) Find 5^x = 6^(x-1)

Log it: x=log5(6^(x-1))

x=(x-1)log5(6)

x=log5(6)x-log5(6)

x-log5(6)x=-log5(6)

x(log5(6)-1)=log5(6)

log5(6)/(log5)(6)-1)=x

x=9.824
