Computer Science A level

storage

A computer uses two types of memory: RAM (Random access memory) and ROM (Read only memory).

RAM is volitile memory which stores the current data of the computer system, like operating system and current running apps.

ROM is non-volitile memory which stores your files and operating system to be loaded on boot. This will be your computer's hard-drive.

There are lots if different ways to store ROM. Some better suited than others.

Solid state

Solid state is where there are no moving parts. This is like flash memory sticks and SD cards.

The advantages

Faster than other memories

Smaller in physical size means lighter

Uneffected by movement as it has no moving parts. Perfect for mobile devices.

Lower power consuption

No noise

The dis-advantages

Expensive per GB

Shorter life-spans

Limited storage capabilities

Optical storage

Optical storage uses visual storage such as CD discs which are read by a laser.

The advantages

Cheap and easy to distribute

light and small

Long life

Lower power consuption

Faster read/write making it better for media

The dis-advantages

Cannot write over data

Easily damaged

Limited storage capabilities

Magnetic

Magnetic uses a read/write head to read/write data from a magnetic disc. This is seen in hard-drives

The advantages

Cheap per GB

Easier to get hold of

Longer lifespan

The dis-advantages

Easily damaged

Big and noisy

Needs alot of power

Slower read/write
