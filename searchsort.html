Computer Science A level

Search and sort

Computer's need to organize data in certain ways to be most efficient in read and write

Bubble sort

Bubble sort goes through an array and switches values when the first value is bigger than the second

It does this till it does not have to make any switches.

The advantages

Easy to understand and implement

Doesn't use additional storage

The dis-advantages

Takes a long time with large arrays.

Insertion sort

This sorts items of an array by moving it down until the value is > it. So 4231, 2 would move before 4, 3 before 4, then 1 before

The advantages

Simple implementation

Quick with shorter algorithms

The dis-advantages

It is slow for large arrays.

Needs large number of element shifts.

Quick sort

Quick sort picks the first value and pivots two lists around it. The smaller values go one side, and bigger values the other.

The system keeps looping till all values have been pivoted around.

The lists are all put together and the array is sorted.

The advantages

Sorts data quickly with huge arrays

The dis-advantages

complex to understand

Merge sort

Sorts data by splitting it in half, into two lists, and repeating this until everything is individual.

It goes through and checks each element and switches them, and puts them into a list of two, then into 4 and keeps increasing

The merge until it is one big sorted array.

The advantages

Quicker than most methods

The dis-advantages

Complex to understand

Uses alot of space to store arrays

Linear search

Goes through an array starting from index 0, and checks every single index till it finds the value it is looking for

The advantages

Easy to understand and write

Doesn't require data to be sorted

The dis-advantages

Bad with large arrays

Binary search

Binary search will go to the middle value and check if the value is < or > than the value it is searching for

It keeps doing this using (n-1/2) to find the value.

The advantages

Quick at finding a value

The dis-advantages

Requires the array to be sorted
