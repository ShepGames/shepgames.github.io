Writing code for websites

You will be asked to write code for websites. There are a few simple rules to this.

HTML

HTML stands for hyper-text-markup-language. It is the 'language' to display webpages.

Basic strucutre:

The basic stucture of a page is as follows:

< !DOCTYPE html >
< head>
everything you need for titles and styles
< /head>
< body >
main webpage
< /body >

Headers

Headers are nice ways of writing titles. we use

< h and the number of the header we want >

Titles

Titles are what we see on the tab of the website

< title > title name < /title >

Style sheets

Styles can be written in in two ways. External files and internal css

< link rel="sytlesheet" href="path/to/file.css" >

< style >
css here
< /style >

Normal text

For normal content use p tags

< p >Text< /p >

Lists

There are two types of lists, ordered and unorderd. Which is numbers and bullet points

< li >
< li >
< /ul >
