Algebra and functions

Definitions:

Range:

Possible y-values

f(x)=x^2 +2 has a domain of {f(x)>=2} as it does not go below 2 on the y-axis

Domain:

Possible x-values

1/(x+1) has a domain of {x!=-1} as -1 would break the equation

Inverse functions:

Get a function and set it equal to y, then switch the y's and x's, ad finally make y the subject.

f(x)=2x+4

y=2x+4

x=2y+4

(x-4)/2=y

Composite functions:

Functions within functions. g(f(x))

g(x)=3x+1

f(x)=2x

g(f(x))=g(2x)=3(2x)+1=6x+1

Inequalities:

Shade the region that satisfies the inequality

y>f(x) is not included, so use a dotted line

Set notation:

A way to express all the valid values of a graph.

{x:2

{x:x<-2}U{x:x>3} shows all the values below -2, and above 3 (not including 2 and 3)

The descriminant:

A way to get a quadratic and work out the roots

ax^2 +bx+c is the typical way of writing an equations

b^2 -4ac is the way of getting the descriminant.

If the answer is < 0 we infer that there are no real roots

If the answer is > 0 we infer there are two real roots, which are unequal

If the answer = 0 then we know that there is one real and equal root.
